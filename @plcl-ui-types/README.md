# PLC UI Types

TypeScript type definitions for the PLC UI library.

## Overview

PLC UI Types provides TypeScript type definitions for the PLC UI desktop library components and utilities.

## Installation

```bash
npm install @plcl/ui-types
```

## Usage

```typescript
import type { 
  AppDefinition, 
  AppWindowProps,
  WindowProps 
} from "@plcl/ui-types";
```

## Development

### Prerequisites

- Node.js >= 18.0.0
- npm >= 9.0.0

### Build

```bash
npm run build
```

### Development

```bash
npm run dev
```

### Lint

```bash
npm run lint
```
